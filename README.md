# The Neural Architect: Akhilesh Pothuri Portfolio

A high-fidelity, cinematic portfolio experience built with **Next.js**, **Three.js**, and **Framer Motion**.

## 🚀 Vision
This portfolio is designed as a "Latent Map" exploring the intersection of Generative AI, Autonomous Agents, and high-performance Data Engineering. It features scrollytelling narratives, interactive 3D skill nebulae, and a project vault of technical prototypes.

## 🛠️ Tech Stack
- **Framework:** Next.js 15 (App Router)
- **3D Engine:** Three.js / @react-three/fiber
- **Animations:** Framer Motion
- **Styling:** Tailwind CSS + Vanilla CSS (Glassmorphism)
- **Deployment:** GitHub Pages (Static Export)

## 📦 Getting Started

### Prerequisites
- Node.js 18+
- npm

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/AKhileshPothuri/Portfolio.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```

## 🌐 Deployment
This project is configured for automated deployment to GitHub Pages via GitHub Actions. Any push to the `main` branch will trigger a build and deploy cycle.
